Python

Python learning It is a clear and legible language which facilitates work and learning for beginners. Python ranks 2nd on the list of the best programming languages, which is another strong argument! The biggest platforms like Spotify, Netflix, Dropbox, Youtube uses this language, and that even makes me more excited to start with it! Programming in Python has many possibilities and applications: Arduino or Raspberry Pi, Cybersecurity, IoT, Marketing (information analysis), Statistics, Machine Learning, AI and Web development. To start this project I just needed Django (Framework) which Python offers. Thanks to this, I started my first real project! When I was coming to finishing writing Backend I understand that I have to start learning and writing basics in HTML5 to get visual output of my job.

HTML 5

What exactly is HTML5? HyperText Markup Language is a so-called hypertext language that describes the contents of the document format, such as a font or paragraph. HTML is used when creating websites. HTML has a so-called skeleton. Thanks to which the order of information and composition of our site should always be the same. When all information and data are applied in our website we need to take care of appearance. In my case I used Bootstrap 4 which is the best framework in my opinion to help you with design!

Bootstrap 4

Bootstrap is a free collection of tools for creating a websites and web applications. It contains HTML and CSS-based design templates for typography, forms, buttons, navigation and other interface components. The primary purpose of adding it to a web project is to apply Bootstrap's choices of color, size, font and layout to that project. As such, the primary factor is whether the developers in charge find those choices to their liking. Once added to a project, Bootstrap provides basic style definitions for all HTML elements.
